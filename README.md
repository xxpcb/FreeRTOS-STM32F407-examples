# 项目结构

## 库与工程模板

- **Template_lib** : STM32F407的库文件(v1.4.0)+FreeRTOS库文件(v9.0.0)
- **Template_noOS** : 基于Keil5.15的STM32F407模板工程(不包含操作系统)
- **Template_TreeRTOS** : 基于Keil5.15的STM32F407+FreeRTOS的模板工程

## 测试例程

- **001_FreeRTOS_LEDtask ** : 任务创建实验，创建两个LED闪烁任务
