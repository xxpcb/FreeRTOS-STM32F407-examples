基于STM32F4 固件库V1.4.0的工程模板

新建工程的时候，请确保Option for target->C/C++选项卡的全部宏定义Define输入框字符串为：STM32F40_41xxx,USE_STDPERIPH_DRIVER

Keil Version:5.15
 
注：此工程将需要用到的库文件(STM32 FreeRTOS)放置在了工程文件外的commonlib文件夹
